# Xenomorph-NodeJS-Server
# Simply NodeJS server with generical router
##########################################

## Get started

### Dependencies

> npm install -g nodemon
> npm install --save-dev nodemon

### Application

> npm install

#### Dev mode

> nodemon app_server.js

#### Production mode

> node app_server.js

Go to http://app.hogwarts.net:9080/get/users
